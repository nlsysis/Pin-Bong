Detailed Description

the illustration of GameObj class The GameObj class represents the basic class contain functions and paraments of the objects of the game. Actually if there are other characters like enemy or bullet can inherit this class. In this program it's the basic class of Player and Orienter.

Parameters
useDescription of the game object's state.
posDescription of the center position of polygon.
moveDescription of the amount of moving in 3 direction.
rotDescription of the angle of rotation in 3 direction.
dirDescription of the direction of player (0-up 1-down 2-left 3-right).
rectDescription of the pointer of polygon.
spriteUVDescription of the texture UV.
widthDescription of the width of the polygon.
heightDescription of the height of the polygon.
alphaDescription of the alpha of the polygon.
CountAnimCount the animation number.It's used if the object has the animation.
TextureThe pointer of texture in Directx9.
tagIf there are different kinds of game object. It can be defined the kind of the object.
